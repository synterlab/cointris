# Cointris 🎮

A retro Game Boy-inspired falling-block puzzle game built with collectible crypto coins from the Orynth ecosystem.

## Play

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- 7 crypto coin tetrominoes (Gold, Silver, Ruby, Emerald, Plasma, Diamond, Nova)
- SRS rotation system with wall kicks
- Ghost piece, hold piece, and 3-piece next queue
- Touch controls (mobile-first) + keyboard support
- Retro Web Audio API sound system
- Combo system and back-to-back bonuses
- 4 unlockable themes
- High score & stats saved in localStorage
- 60 FPS canvas rendering
